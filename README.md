# SMTP Mail System App

SMTP Mail System Desktop App / SMTP Posta Sistemi Masaüstü Uygulaması

# Modules / Modüller

* smtplib
* ssl
* email
* PyQt5

# Setup / Kurulum

```css

pip install smtplib
pip install ssl
pip install email
pip install PyQt5

```

#### Python Version 
      3.8.5
